## カラーパレット

ロゴ（CRAFT マスコット）の3色をベースに設定しています。

| 変数名 | カラーコード | 用途 |
|--------|-------------|------|
| `--yellow` | `#F7C530` | メインアクセント・ボタン |
| `--red` | `#E5402E` | サブアクセント・アイコン |
| `--blue` | `#3D72B4` | サブアクセント・装飾 |
| `--ink` | `#2B1B10` | 本文・ボーダー |
| `--cream` | `#FFF8E9` | 背景ベース |
